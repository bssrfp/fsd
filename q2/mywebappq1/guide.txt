🔹 Step 3: Initialize Git Repository
git init
🔹 Step 4: Add Files to Staging Area
git add .
🔹 Step 5: Commit Files
git commit -m "Initial commit with 5-page web app"
🔹 Step 6: Create Remote Repository on GitHub
Go to 👉 GitHub
Click New Repository
Name: mywebapp
Click Create Repository
🔹 Step 7: Connect Local Repo to GitHub
git remote add origin https://github.com/your-username/mywebapp.git
🔹 Step 8: Push Code to GitHub
git branch -M main
git push -u origin main
🔄 Git Operations (Very Important for Viva)
🔹 1. Push

Uploads local commits to GitHub

git push origin main

👉 Use when you want to upload changes

🔹 2. Pull

Fetch + Merge from remote to local

git pull origin main

👉 Updates your local repo

🔹 3. Fetch

Only downloads changes (no merge)

git fetch origin

👉 Safe way to check updates
